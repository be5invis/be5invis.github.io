简单来说就是，我终于抛弃三年前自己写的脚本，换用 Hexo 啦！给 Hexo 写 Youki 的 renderer 看上去也没有那么困难，当然为了 hexo s，只能放弃 Youki 的「单次执行加载模块」，每次都解释一遍——好在消耗不算大。

但是！现在终于不用每写一段就 make 一下网站然后手动刷新浏览器了，BrowserSync 真是个好帮手。

现在这个主题的皮肤大体上是和 函子风云录 网站用的是相同的，都是修改自我一个用了很久的 Typora 主题。当然了，Typeof.net 所用的版本多了一些内容比如公式排版（ykeqn）专用的一些样式等等。

看来我又可以去拾起来写文章的兴趣了呢。

Finis.
